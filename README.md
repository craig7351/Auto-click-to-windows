# AfkKeeper — 視窗定時送鍵工具

對**指定的已開啟視窗**，每隔自訂時間送出一個按鍵（預設空白鍵）。

## 特色

- **不干擾你的操作**：用 `GetLastInputInfo` 偵測閒置，只在你手停下來的空檔才送鍵，送完立刻把焦點還給你原本的視窗（約 0.2 秒）
- **遊戲收得到**：用 `SendInput` + scan code，而非 `PostMessage`（Roblox 等用 DirectInput 的遊戲可以收到）
- **防規律偵測**：間隔支援 ± 隨機秒數抖動
- **單一 exe，無需安裝**：使用 Windows 內建 .NET Framework 4.x，雙擊即用（約 16 KB）
- 可縮到**系統匣**背景執行

## 下載

直接下載 [AfkKeeper.exe](AfkKeeper.exe)，雙擊執行，無需安裝任何環境。

## 使用方式

1. 開啟目標程式（例如 Roblox）
2. 執行 AfkKeeper，從下拉選單選擇目標視窗（自動嘗試選中 Roblox）
3. 設定間隔秒數（預設 780 秒 = 13 分）、按鍵、隨機抖動
4. 勾選「只在我閒置時送鍵」（推薦）
5. 按「▶ 開始」

## 重新編譯

```bat
build.bat
```

使用 Windows 內建 `csc.exe`，不需安裝 .NET SDK。

## 注意

本工具為純外部輸入模擬（不注入 DLL、不讀寫遊戲記憶體），但仍可能違反特定遊戲服務條款中關於掛機的規定。請自行評估風險，建議僅供個人使用。
