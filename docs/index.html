定時對指定視窗送出按鍵，只在你手停下來的空檔執行——不干擾你、遊戲收得到。

⬇️ 下載 AfkKeeper.exe 📂 GitHub 原始碼

✨ 功能特色

🎯

指定視窗送鍵

下拉選擇任何已開啟的視窗，不影響其他程式。自動嘗試選中 Roblox。

🕐

閒置偵測

只在你手沒在動的空檔送鍵，送完立刻把焦點還給你原本的視窗，干擾約 0.2 秒。

🎮

遊戲真的收得到

用 SendInput + scan code，而非 PostMessage。Roblox 等用 DirectInput 的遊戲可以收到。

🎲

隨機抖動防偵測

間隔支援 ± 隨機秒數，避免毫秒不差的規律被伺服器判定為機器人。

🗂️

多按鍵選擇

空白鍵、W、上/下箭頭、E、數字 0，對應各遊戲的跳/移動/互動鍵。

🔔

縮到系統匣

最小化後隱入系統匣繼續背景執行，雙擊圖示還原視窗。

📦

單一 exe，無需安裝

.NET Framework 4.x 是 Windows 10/11 內建，雙擊即用，不用安裝任何環境。

🔄

重開遊戲也找得到

依程序名重新定位視窗 handle，遊戲重啟後不需要重新設定。

🚀 使用方式

  1. 下載並執行 AfkKeeper.exe

    不需安裝，雙擊即可開啟。

  2. 開啟目標遊戲（例如 Roblox）

    從下拉選單選擇目標視窗，按「刷新」重新列舉。

  3. 設定間隔與按鍵

    間隔預設 780 秒（13 分），隨機抖動 ±120 秒。按鍵預設空白鍵。

  4. 勾選「只在我閒置時送鍵」（推薦）

    到期後等你手停 3 秒才送；若等超過 90 秒仍會強制送出，避免被踢。

  5. 按「▶ 開始」，最小化到系統匣

    背景繼續執行，紀錄欄會顯示每次送鍵時間。
